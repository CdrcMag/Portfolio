Prototype GD : Spectroscopie astronomique

Description

Ce projet est un prototype de serious game orienté vers l'apprentissage des notions fondamentales de spectroscopie astronomique. Ce projet a été utilisé pour apprendre à transmettre un important volume d'informations sans surcharger le joueur. Le jeu est découpé en 2 parties. La première introduit certaines notions à travers un cours donné par le personnage robot. La seconde partie est un mini jeu d'évaluation, dans lequel le joueur devra utiliser les informations précédemment acquises.
