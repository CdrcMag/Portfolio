Hyperloop

Description

Hyperloop est un jeu mobile d'aracade. Il a été développé en parallèle d'un projet de développement d'outil traitant le daltonisme. Ce jeu est donc basé sur les couleurs, afin de mettre en avant les problèmatiques que les joueurs atteints peuvent rencontrer.
